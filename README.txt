	2) Criar um teste/exame onde as respostas são checadas a partir de um gabarito no servidor. 
    O teste deve ser objetivo, fazendo uso de múltipla escolha (4 ou 5 opções), verdadeiro ou falso, seleção de sentenças verdadeiras, etc. 
    As questões devem ser selecionadas aleatoriamente de um banco de questões (em um ficheiro, por exemplo) também no servidor. 
    O frontend, interface com o utilizador, deve apresentar o formulário com as questões e possibilitar que elas sejam avaliadas.
